# Flutter Modern Chat UI 💬

A modern, beautiful, and fully customizable Flutter chat UI package designed for messaging applications, social apps, and real-time communication platforms.

Built with clean architecture, reusable widgets, responsive layouts, and smooth modern UI components to help developers create professional chat applications faster.

---

# ✨ Features

- 🎨 Modern and clean chat interface
- 💬 Beautiful customizable chat bubbles
- 📱 Responsive UI for all screen sizes
- 🧩 Reusable Flutter widgets
- 📂 Sidebar navigation support
- 📨 Message input component
- 👤 Chat tile and user list support
- 🖥️ Cross-platform support
- ⚡ Lightweight and easy to integrate
- 🛠️ Developer-friendly structure

---

# 📸 Included Components

This package includes:

- Home Screen
- Chat Screen
- Sidebar Navigation
- Chat Bubble Widget
- Chat Tile Widget
- Message Input Widget
- Chat Header
- Chat List Components
- Theme Support

# 🚀 Getting Started

## Installation

Add the package to your `pubspec.yaml` file:

```yaml
dependencies:
  flutter_mordern_chat_ui: ^1.0.1
```

Then run:

```bash
flutter pub get
```

---

# 📦 Import Package

```dart
import 'package:flutter_mordern_chat_ui/modern_chat_ui.dart';
```

---

# 💻 Basic Usage

```dart
import 'package:flutter/material.dart';
import 'package:flutter_mordern_chat_ui/modern_chat_ui.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      debugShowCheckedModeBanner: false,
      home: HomeScreen(),
    );
  }
}
```
